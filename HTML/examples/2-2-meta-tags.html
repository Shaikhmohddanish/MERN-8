What Are Meta Tags?

Meta tags are snippets of HTML code that provide metadata about a webpage. They don't affect the visible content but provide information about the page to browsers and search engines.

Note: Most meta tags are placed within the <head> section of an HTML document.

Essential Meta Tags

These meta tags should be included in virtually every HTML document:

Character Encoding

<meta charset="UTF-8">

Specifies the character encoding for the HTML document. UTF-8 is recommended as it supports characters from all languages.

Viewport Settings

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Controls how the page is scaled and displayed on different devices, essential for responsive design.

Description

<meta name="description" content="A concise summary of the page content.">

Provides a brief summary of the page that may appear in search engine results.

SEO Meta Tags

These meta tags help improve search engine optimization:

Meta Tag Purpose
<meta name="description" content="..."> Page description for search results
<meta name="keywords" content="..."> Keywords related to the page (less important now)
<meta name="robots" content="index, follow"> Instructions for search engine crawlers
<link rel="canonical" href="..."> Specifies the preferred URL for the page
<meta name="author" content="..."> Identifies the page author

Robots Directives

The "robots" meta tag can include these values:

  • index: Allow indexing of the page
  • noindex: Prevent indexing of the page
  • follow: Allow following links on the page
  • nofollow: Prevent following links on the page
  • noarchive: Prevent caching of the page
  • nosnippet: Prevent displaying a snippet in search results
  • notranslate: Prevent translation of the page
  • noimageindex: Prevent indexing of images on the page
<meta name="robots" content="index, follow, max-snippet:-1, max-image-preview:large, max-video-preview:-1">

Social Media Meta Tags

Open Graph Protocol (Facebook, LinkedIn, Pinterest)

Open Graph meta tags control how your content appears when shared on Facebook and many other platforms:

<meta property="og:type" content="article">
<meta property="og:title" content="Your Page Title">
<meta property="og:description" content="Your page description">
<meta property="og:image" content="https://example.com/image.jpg">
<meta property="og:url" content="https://example.com/page">
<meta property="og:site_name" content="Your Site Name">

Twitter Card (Twitter)

Twitter Card meta tags control how your content appears when shared on Twitter:

<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:site" content="@yourusername">
<meta name="twitter:title" content="Your Page Title">
<meta name="twitter:description" content="Your page description">
<meta name="twitter:image" content="https://example.com/image.jpg">

Twitter card types include:

  • summary: Standard card with small image
  • summary_large_image: Card with large image
  • app: Card for mobile apps
  • player: Card for video/audio players

Technical and HTTP-Equiv Meta Tags

These meta tags provide instructions to the browser about how to handle the page:

Page Refresh or Redirect

<meta http-equiv="refresh" content="5;url=https://example.com/new-page">

Redirects to a new URL after 5 seconds. Use with caution as it can confuse users.

X-UA-Compatible

<meta http-equiv="X-UA-Compatible" content="IE=edge">

Ensures Internet Explorer uses the latest rendering engine.

Cache Control

<meta http-equiv="cache-control" content="no-cache">
<meta http-equiv="expires" content="0">

Controls how the browser caches the page.

Mobile-Specific Meta Tags

These meta tags enhance the experience on mobile devices:

Apple Mobile Web App

<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-status-bar-style" content="black">
<meta name="apple-mobile-web-app-title" content="App Title">

Controls how the page behaves when saved to the home screen on iOS devices.

Theme Color

<meta name="theme-color" content="#3498db">

Controls the color of the browser UI on some mobile devices, particularly Android Chrome.

Telephone Format Detection

<meta name="format-detection" content="telephone=no">

Prevents browsers from automatically detecting and formatting phone numbers as clickable links.

Specificity and the Cascade

Specificity Examples

This paragraph has multiple styles applied with different specificity:
- Element selector (p)
- Class selector (.spec-example)
- ID selector (#spec-id)
- Inline style (style attribute)

/* Least specific */
p {
  color: blue;
}

/* More specific */
.spec-example {
  color: green;
}

/* Even more specific */
#spec-id {
  color: red;
}

/* Most specific (inline) */
<p style="border-bottom: 2px dashed purple;">

The color of the text is determined by the most specific selector (ID selector in this case).

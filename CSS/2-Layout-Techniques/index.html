Responsive Layouts

Media Queries

Resize the browser window to see this change 

.responsive-element {
    padding: 20px;
    background-color: #e6f2ff;
    border: 1px solid #0066cc;
    text-align: center;
}

/* Small screens */
@media (max-width: 600px) {
    .responsive-element {
        background-color: #ffe6e6;
        border-color: #cc0066;
    }
}

/* Medium screens */
@media (min-width: 601px) and (max-width: 1024px) {
    .responsive-element {
        background-color: #e6ffe6;
        border-color: #66cc66;
    }
}

/* Large screens */
@media (min-width: 1025px) {
    .responsive-element {
        background-color: #e6f2ff;
        border-color: #0066cc;
    }
}

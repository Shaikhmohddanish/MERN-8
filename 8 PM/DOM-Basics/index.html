Understanding the DOM

The Document Object Model (DOM) is a programming interface for HTML documents

DOM Structure

The DOM represents an HTML document as a tree of nodes:

document
html
head
title
body
header
h1
p
section

Accessing the DOM

The entry point to the DOM is the document object

Open the browser console and try:

console.log(document);

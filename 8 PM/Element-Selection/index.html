JavaScript Element Selection Methods

Learn the various ways to select DOM elements with JavaScript

Selection Methods Overview

getElementById

Selects a single element by its ID attribute

document.getElementById('uniqueId')

getElementsByClassName

Selects multiple elements by their class name

document.getElementsByClassName('className')

getElementsByTagName

Selects multiple elements by their tag name

document.getElementsByTagName('div')

querySelector

Selects the first element that matches a CSS selector

document.querySelector('.class #id')

querySelectorAll

Selects all elements that match a CSS selector

document.querySelectorAll('div.className')

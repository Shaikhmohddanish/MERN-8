DOM Manipulation Techniques

Content Manipulation

This is sample text content

Methods Explained:

textContent
Gets or sets the text content of an element and all its descendants
innerHTML
Gets or sets the HTML content inside the element (parses HTML)
innerText
Gets or sets the visible text content of an element (affected by CSS)

Attribute Manipulation

Hover over me to see the title attribute

Methods Explained:

getAttribute(name)
Gets the value of the specified attribute
setAttribute(name, value)
Sets the value of the specified attribute
removeAttribute(name)
Removes the specified attribute
hasAttribute(name)
Returns true if the element has the specified attribute

Methods Explained:

element.style.property
Sets inline CSS styles directly
element.classList.add()
Adds a CSS class to an element
element.classList.remove()
Removes a CSS class from an element
element.classList.toggle()
Toggles a CSS class (adds if not present, removes if present)

Create & Delete Elements

Element Playground

Methods Explained:

document.createElement()
Creates a new element node
parent.appendChild()
Adds a node as the last child of a parent
parent.prepend()
Adds a node as the first child of a parent
parent.insertBefore()
Inserts a node before a reference node
parent.replaceChild()
Replaces a child node with a new node
node.cloneNode()
Creates a copy of a node
node.remove()
Removes a node from the DOM
